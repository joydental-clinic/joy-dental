# 연세조이치과 홈페이지 운영 가이드

> 이 문서는 홈페이지 관리 시스템을 처음 사용하는 분도 혼자 따라할 수 있도록 작성되었습니다.
> 단계를 순서대로 따라가면 됩니다.

---

## 0. 홈페이지를 구성하는 서비스 안내

홈페이지는 세 가지 외부 서비스가 서로 연결되어 운영됩니다. 평소에는 아래 내용을 몰라도 되지만, 문제가 생기거나 계정 정보가 필요할 때 참고하세요.

**모든 서비스의 로그인 계정은 동일합니다.**

> 계정 이메일: **joydental-host@outlook.com**

---

### 세 서비스의 역할

```
콘텐츠 입력 (Sanity)
        ↓
코드 보관 (GitHub)
        ↓
홈페이지 게시 (Cloudflare)
```

---

### Sanity — 콘텐츠 관리 (일상 업무)

**역할:** 공지사항, 칼럼, 진료시간, 의료진 등 홈페이지에 표시되는 **내용**을 관리하는 곳입니다.

평소에 가장 자주 접속하게 될 서비스입니다. 워드프레스나 블로그 관리자 화면과 비슷하다고 생각하면 됩니다.

- 접속 주소: `https://ysjoy-dental-clinic.sanity.studio/`
- 로그인: `joydental-host@outlook.com`
- 이 문서의 2장~9장이 Sanity 사용 방법입니다.

---

### GitHub — 코드 보관소 (건드릴 필요 없음)

**역할:** 홈페이지를 만드는 **소스 코드**가 저장된 창고입니다. 홈페이지의 디자인, 구조, 기능이 여기에 저장되어 있습니다.

> 일반적인 운영 중에는 접속할 필요가 없습니다. 개발자에게 홈페이지 수정을 의뢰할 때 사용하는 곳입니다.

- 접속 주소: `https://github.com`
- 로그인: `joydental-host@outlook.com`

---

### Cloudflare — 홈페이지 서버 및 도메인 (건드릴 필요 없음)

**역할:** 쉽게 말해 홈페이지가 실제로 돌아가는 **컴퓨터(서버)를 빌려주는 곳**입니다. 또한 방문자가 주소창에 입력하는 **도메인(`ysjoy.com`)도 여기서 관리**합니다.

병원 홈페이지를 "간판"에 비유하면, Cloudflare는 그 간판이 걸려 있는 **건물**입니다. 건물이 있어야 간판을 걸 수 있고, 주소도 유지됩니다. 평소에는 신경 쓸 필요가 없지만, 건물이 무너지면(서버 장애) 간판도 보이지 않게 됩니다.

한 가지 더 — 도메인은 매년 **사용료를 갱신**해야 합니다. 갱신하지 않으면 주소가 사라져 홈페이지에 접속할 수 없게 됩니다. **도메인 만료 안내 이메일이 오면 반드시 개발자에게 알려주세요.**

> 일반적인 운영 중에는 접속할 필요가 없습니다. 홈페이지가 갑자기 안 열리거나 도메인 관련 이메일이 오면 개발자에게 문의하세요.

- 접속 주소: `https://dash.cloudflare.com`
- 로그인: `joydental-host@outlook.com`

---

### 정리: 언제 어디에 접속하나요?

| 상황 | 접속할 서비스 |
|---|---|
| 공지사항·칼럼 작성 / 진료시간·의료진 수정 | **Sanity** (일상 업무) |
| 홈페이지 디자인·기능 변경 의뢰 | GitHub (개발자 담당) |
| 도메인 문제, 홈페이지 다운 | Cloudflare (개발자 담당) |

## 1. 시작하기 — 관리 화면 접속

### 주소

브라우저(크롬 권장)에서 아래 주소로 접속합니다.

```
https://ysjoy-dental-clinic.sanity.studio/
```

### 로그인 방법

1. 접속하면 로그인 화면이 나타납니다.
2. **Continue with email**을 클릭합니다.
3. 등록된 이메일 주소를 입력하고 **Continue**를 클릭합니다.
4. 해당 이메일로 인증 링크가 발송됩니다.
5. 이메일을 열어 **로그인 링크**를 클릭하면 자동으로 접속됩니다.

> 인증 이메일이 보이지 않으면 스팸 메일함을 확인하세요.

### 화면 기본 구성

로그인 후 화면 왼쪽에 메뉴 목록이 표시됩니다.

| 메뉴 이름 | 설명 |
|---|---|
| 공지사항 | 휴진, 진료 안내 등 공지 관리 |
| 칼럼 | 블로그 형태의 진료 정보 글 관리 |
| 진료시간 | 요일별 진료시간 관리 |
| 기본정보 | 전화번호, 주소 등 병원 정보 관리 |
| 의료진 | 의사·직원 프로필 관리 |
| 히어로 슬라이드 | 메인 화면 상단 이미지 관리 |

## 2. 콘텐츠 반영 시점 안내 ⚠️

> **반드시 읽어주세요.** 내용을 저장(발행)한 뒤 실제 홈페이지에 반영되는 데 걸리는 시간이 항목마다 다릅니다.

| 항목 | 홈페이지 반영 시간 |
|---|---|
| **공지사항** | 발행 즉시 반영 |
| 칼럼(블로그) | 최대 **1분** 후 반영 |
| 진료시간 | 최대 **1분** 후 반영 |
| 병원 기본정보 | 최대 **1분** 후 반영 |
| 의료진 | 최대 **1분** 후 반영 |
| 메인 이미지 | 최대 **1분** 후 반영 |

반영이 안 된 것처럼 보이면 브라우저에서 새로고침(F5 또는 Ctrl+R)을 해보세요.

## 3. 공지사항 관리 — 가장 자주 쓰는 기능

### 3-1. 새 공지 작성

1. 왼쪽 메뉴에서 **공지사항**을 클릭합니다.
2. 오른쪽 상단의 **+ 새 문서** 버튼을 클릭합니다.
3. **제목** 칸에 공지 제목을 입력합니다.
4. **주소(URL)** 칸이 자동으로 채워집니다. 그대로 두면 됩니다.

   > 주소(URL)는 한 번 정해지면 나중에 바꾸지 않는 것이 좋습니다. 바꾸면 이전 링크가 연결되지 않을 수 있습니다.

5. **카테고리**를 선택합니다. (아래 설명 참고)
6. 필요하면 **상단 고정** 체크박스를 체크합니다.
7. **내용** 칸에 공지 본문을 작성합니다.
8. 작성이 끝나면 오른쪽 상단의 **발행(Publish)** 버튼을 클릭합니다.

> **발행(Publish) 버튼을 누르지 않으면 홈페이지에 게시되지 않습니다.**

### 3-2. 카테고리 선택

| 카테고리 | 사용 예시 |
|---|---|
| 일반 | 그 외 일반 공지 |
| 진료안내 | 진료 변경, 특별 진료 안내 |
| 휴진안내 | 휴일 진료, 임시 휴진 안내 |

### 3-3. 상단 고정 (공지를 목록 맨 위에 표시)

- **상단 고정** 체크박스를 체크하면 해당 공지가 목록 맨 위에 표시됩니다.
- 중요한 공지(예: 장기 휴진 안내)에 사용합니다.
- 공지가 끝나면 체크를 해제하고 다시 **발행** 버튼을 누릅니다.

### 3-4. 기존 공지 수정

1. **공지사항** 메뉴를 클릭합니다.
2. 수정할 공지의 제목을 클릭합니다.
3. 내용을 수정합니다.
4. **발행(Publish)** 버튼을 클릭합니다.

### 3-5. 공지 삭제

1. 삭제할 공지를 엽니다.
2. 오른쪽 상단의 **...** (더보기) 버튼을 클릭합니다.
3. **삭제**를 클릭하고 확인합니다.

> 삭제한 공지는 복구할 수 없습니다. 신중하게 진행해주세요.

## 4. 칼럼(블로그) 관리

칼럼은 교정, 임플란트, 일반 진료 정보를 알리는 블로그 형태의 글입니다.

### 4-1. 새 칼럼 작성

1. 왼쪽 메뉴에서 **칼럼**을 클릭합니다.
2. **+ 새 문서** 버튼을 클릭합니다.
3. **제목**을 입력합니다.
4. **주소(URL)**는 자동으로 채워집니다. 그대로 두면 됩니다.

   > 주소(URL)는 한 번 정해지면 나중에 바꾸지 마세요.

5. **카테고리**를 선택합니다.

### 4-2. 카테고리 선택

| 카테고리 | 용도 |
|---|---|
| 교정 | 치아 교정 관련 정보 |
| 임플란트 | 임플란트 관련 정보 |
| 일반 | 그 외 진료 정보 |

### 4-3. 썸네일 이미지 업로드

썸네일은 칼럼 목록에서 글 옆에 표시되는 대표 이미지입니다.

1. **썸네일 이미지** 항목의 **이미지 업로드** 버튼을 클릭합니다.
2. 컴퓨터에 저장된 이미지 파일을 선택합니다.
3. 이미지가 업로드되면 미리보기가 표시됩니다.

> 이미지는 JPG 또는 PNG 파일을 사용하세요. 너무 작은 이미지는 흐릿하게 보일 수 있습니다.

### 4-4. 본문에 이미지 넣기

1. 본문 입력 칸에서 이미지를 넣을 위치에 커서를 놓습니다.
2. 본문 입력 칸 상단의 **이미지 아이콘**(산 모양)을 클릭합니다.
3. 이미지 파일을 선택하면 본문 안에 이미지가 삽입됩니다.

### 4-5. 발행

1. 모든 내용 작성 후 **발행(Publish)** 버튼을 클릭합니다.
2. 최대 1분 후 홈페이지 칼럼 목록에 표시됩니다.

## 5. 진료시간 변경

> 진료시간은 하나의 문서로 관리됩니다. 요일별로 개별 항목을 수정합니다.

### 5-1. 진료시간 수정 화면 열기

1. 왼쪽 메뉴에서 **진료시간**을 클릭합니다.
2. 진료시간 문서가 자동으로 열립니다. (새 문서 만들기 불필요)

### 5-2. 요일별 시간 변경

1. 변경할 요일 항목을 찾습니다.
2. **시간** 칸을 클릭하고 새로운 시간을 입력합니다.
   예: `09:00 - 18:00`
3. 오른쪽 상단의 **발행(Publish)** 버튼을 클릭합니다.

### 5-3. 휴진일 설정

1. 휴진으로 설정할 요일 항목을 찾습니다.
2. **휴진** 체크박스를 체크합니다.
3. **발행(Publish)** 버튼을 클릭합니다.

휴진이 끝나면 체크를 해제하고 다시 **발행** 버튼을 누르세요.

### 5-4. 강조 표시 설정

특정 요일(예: 점심시간 다른 날, 특별 연장 진료)을 눈에 띄게 표시하려면:

1. 해당 요일 항목에서 **강조** 체크박스를 체크합니다.
2. **발행(Publish)** 버튼을 클릭합니다.

---

## 6. 병원 기본정보 변경

전화번호, 이메일, 주소 등 병원의 기본 정보를 수정합니다.

### 수정 방법

1. 왼쪽 메뉴에서 **기본정보**를 클릭합니다.
2. 기본정보 문서가 자동으로 열립니다.
3. 변경할 항목을 클릭하고 수정합니다.

| 항목 | 설명 |
|---|---|
| 전화번호 | 병원 대표 전화번호 |
| 이메일 | 병원 대표 이메일 |
| 주소 | 병원 주소 |

4. 수정 후 **발행(Publish)** 버튼을 클릭합니다.

---

## 7. 의료진 관리

### 7-1. 새 의료진 추가

1. 왼쪽 메뉴에서 **의료진**을 클릭합니다.
2. **+ 새 문서** 버튼을 클릭합니다.
3. 아래 항목을 입력합니다.

| 항목 | 설명 |
|---|---|
| 이름 | 의사·직원 이름 |
| 직책 | 예: 대표원장, 부원장, 치위생사 |
| 전문 분야 | 예: 교정, 임플란트 |
| 사진 | 프로필 사진 업로드 |
| 소개 | 학력, 경력 등 |

4. **발행(Publish)** 버튼을 클릭합니다.

### 7-2. 의료진 정보 수정

1. **의료진** 메뉴에서 수정할 이름을 클릭합니다.
2. 내용을 수정합니다.
3. **발행(Publish)** 버튼을 클릭합니다.

### 7-3. 의료진 순서 변경

1. **의료진** 메뉴에서 순서를 바꿀 의사·직원의 문서를 엽니다.
2. **순서** 항목의 숫자를 변경합니다.
   (숫자가 낮을수록 앞에 표시됩니다. 예: 1 → 가장 먼저)
3. **발행(Publish)** 버튼을 클릭합니다.

---

## 8. 히어로 슬라이드 관리

히어로 슬라이드는 홈페이지 최상단에 자동으로 넘어가는 대형 이미지입니다.

### 8-1. 메인 이미지 교체

1. 왼쪽 메뉴에서 **히어로 슬라이드**를 클릭합니다.
2. 교체할 슬라이드 항목을 클릭합니다.
3. 기존 이미지를 클릭하고 **제거** 버튼을 클릭합니다.
4. **이미지 업로드** 버튼을 클릭하여 새 이미지를 업로드합니다.
5. **발행(Publish)** 버튼을 클릭합니다.

> 메인 이미지는 가로가 넓은 이미지(가로 1920px 이상 권장)를 사용해야 깔끔하게 표시됩니다.

### 8-2. 슬라이드 순서 변경

1. 순서를 바꿀 슬라이드 항목을 엽니다.
2. **순서** 항목의 숫자를 변경합니다.
   (숫자가 낮을수록 먼저 표시됩니다.)
3. **발행(Publish)** 버튼을 클릭합니다.

## 9. 자주 하는 실수 & 주의사항

### ⚠️ 발행(Publish) 버튼을 누르지 않으면 홈페이지에 반영되지 않습니다

> 내용을 작성하거나 수정한 뒤에는 반드시 **발행(Publish)** 버튼을 눌러야 합니다.
> 버튼을 누르지 않으면 관리 화면에는 보이지만 홈페이지에는 표시되지 않습니다.

발행 전에는 문서 상태가 **"초안(Draft)"** 으로 표시됩니다.
발행 후에는 **"게시됨(Published)"** 으로 바뀝니다.

---

### ⚠️ 주소(URL)는 한 번 정하면 바꾸지 마세요

> 공지사항이나 칼럼의 주소(URL)를 나중에 변경하면, 기존에 공유된 링크가 연결되지 않습니다.
> 처음 작성할 때 자동으로 채워지는 주소를 그대로 사용하세요.

---

### ⚠️ 이미지 파일 이름에 한글 사용 금지

> 이미지 파일 이름에 한글이 포함되어 있으면 업로드가 실패할 수 있습니다.
> 파일 이름은 영문과 숫자만 사용하세요. (예: `photo-2024.jpg`)

### 자주 묻는 질문

**Q. 수정했는데 홈페이지에 반영이 안 됩니다.**
A. 발행(Publish) 버튼을 눌렀는지 확인하세요. 눌렀다면 브라우저를 새로고침(F5)하고 최대 1분 기다려주세요.

**Q. 이메일 로그인 링크가 오지 않습니다.**
A. 스팸 메일함을 확인하세요. 그래도 없으면 잠시 후 다시 시도하거나 담당자에게 문의하세요.

**Q. 잘못 수정했는데 되돌릴 수 있나요?**
A. 발행(Publish) 전이면 화면을 닫으면 됩니다. 발행 후라면 내용을 다시 수정하고 재발행하세요.

**Q. 삭제한 글을 복구할 수 있나요?**
A. 삭제한 문서는 복구할 수 없습니다. 삭제 전에 신중하게 확인해주세요.

---

*문의사항은 홈페이지 개발 담당자에게 연락해주세요.*
